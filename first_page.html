All About HyperLinks

This is where we are going to learn all about how to link one page to another.

Hyperlinks allow users to move from one web page to another and share information across different machines and systems

How to create a hyperlink on a web page?

The anchor tag is used to create a hyperlink on a web page.
The anchor tag is a container tag, meaning it requires both opening and closing tags

To create a hyperlink, you need "href" to add the destination URL.
"href stands for HyperText reference".
